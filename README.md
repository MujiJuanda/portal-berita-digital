# Portal Berita Digital 📰

## 📖 About

Portal Berita Digital is a web-based news platform designed to publish, manage, and organize digital news content. The project focuses on delivering a clean, responsive, and user-friendly interface for readers while providing efficient content management features.

This project was developed as part of my learning journey in web development and user interface design.

---

## 🎯 Objectives

- Provide an online platform for publishing news.
- Deliver a responsive and user-friendly reading experience.
- Organize news articles into categories.
- Improve accessibility of digital news content.
